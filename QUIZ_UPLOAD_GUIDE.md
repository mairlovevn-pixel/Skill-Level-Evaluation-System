# Written Test Quiz 업로드 가이드

## 지원하는 엑셀 파일 형식

### 형식 1: 영문 컬럼명 (Process ID 포함)
```
| Process ID | Question | Option A | Option B | Option C | Option D | Correct Answer |
|------------|----------|----------|----------|----------|----------|----------------|
| 1          | 질문1    | 옵션A    | 옵션B    | 옵션C    | 옵션D    | A              |
| 2          | 질문2    | 옵션A    | 옵션B    | 옵션C    | 옵션D    | B              |
```

**특징:**
- Process ID를 각 행에 포함
- 여러 프로세스의 퀴즈를 한 번에 업로드 가능
- 프로세스 선택 불필요

### 형식 2: 한국어 컬럼명 (프로세스 선택 필요)
```
| 번호 | 질문 | 1) | 2) | 3) | 4) | 정답 | 배점 |
|------|------|----|----|----|----|------|------|
| 1    | 질문1| 옵션A | 옵션B | 옵션C | 옵션D | A | 4 |
| 2    | 질문2| 옵션A | 옵션B | 옵션C | 옵션D | B | 4 |
```

**특징:**
- 한국어 컬럼명 사용
- 웹 페이지에서 프로세스 선택 필요
- 모든 퀴즈가 선택한 프로세스로 등록됨
- 정답은 A, B, C, D 형식으로 입력

## 업로드 방법

### 형식 1 사용 시
1. Quiz 등록 페이지 접속
2. 엑셀 파일 선택
3. '퀴즈 업로드' 버튼 클릭

### 형식 2 사용 시 (예: CNC_Plasma_Quiz_Full.xlsx)
1. Quiz 등록 페이지 접속
2. **프로세스 선택** (예: CNC Plasma)
3. 엑셀 파일 선택
4. '퀴즈 업로드' 버튼 클릭

## 정답 형식

### 허용되는 정답 형식
- `A`, `B`, `C`, `D` (대문자)
- `a`, `b`, `c`, `d` (소문자, 자동으로 대문자로 변환)

### 주의사항
- 정답은 반드시 A, B, C, D 중 하나여야 합니다
- 빈 칸이나 잘못된 형식은 오류 메시지를 표시합니다

## 예제 파일

### CNC_Plasma_Quiz_Full.xlsx
- 형식: 한국어 컬럼명 (번호, 질문, 1), 2), 3), 4), 정답, 배점)
- 프로세스: CNC Plasma 선택 필요
- 총 25문항

## 오류 해결

### "프로세스를 선택해주세요"
- 형식 2 파일을 사용할 때 프로세스를 선택하지 않으면 발생
- 해결: 페이지 상단의 프로세스 선택 드롭다운에서 프로세스 선택

### "지원하지 않는 엑셀 파일 형식입니다"
- 컬럼명이 형식 1, 2 모두와 일치하지 않을 때 발생
- 해결: 컬럼명을 확인하고 형식에 맞게 수정

### "필수 항목이 누락되었습니다"
- 질문, 옵션 A, 옵션 B, 정답 등 필수 항목이 비어있을 때 발생
- 해결: 각 행의 필수 항목을 모두 입력

### "정답이 올바르지 않습니다"
- 정답이 A, B, C, D가 아닌 다른 값일 때 발생
- 해결: 정답을 A, B, C, D 중 하나로 수정
