각 작업자는 다음 3가지 성과 등급 중 하나로 무작위 분류됩니다:

1. **우수 (Excellent)** - 20%
   - Level 분포: Level 3 (10%), Level 4 (40%), Level 5 (50%)
   
2. **양호 (Good)** - 50%
   - Level 분포: Level 2 (20%), Level 3 (50%), Level 4 (30%)
   
3. **평균 (Average)** - 30%
   - Level 분포: Level 1 (20%), Level 2 (50%), Level 3 (30%)

## 🗑️ 테스트 데이터 삭제 방법

### 방법 1: 삭제 스크립트 실행 (권장)
```bash
cd /home/user/webapp
./delete_test_data.sh
```

### 방법 2: 직접 SQL 명령 실행
```bash
cd /home/user/webapp
npx wrangler d1 execute webapp-production --local --command="DELETE FROM supervisor_assessments;"
```

### 방법 3: 웹 브라우저에서 확인 후 삭제
1. 대시보드에서 현재 데이터 확인
2. 위 방법 1 또는 2로 삭제
3. 브라우저 새로고침하여 데이터 삭제 확인

## ⚠️ 주의사항

1. **Written Test 데이터는 삭제되지 않습니다**
   - 이 스크립트는 Supervisor Assessment 데이터만 삭제합니다
   - Written Test 결과는 그대로 유지됩니다

2. **로컬 데이터베이스만 영향받습니다**
   - `--local` 플래그로 로컬 D1 데이터베이스만 수정합니다
   - 프로덕션 데이터베이스는 영향받지 않습니다

3. **데이터 백업은 자동으로 되지 않습니다**
   - 필요시 삭제 전에 Excel 다운로드로 백업하세요
