# 작업자 현황 업로드 가이드

## 지원하는 엑셀 파일 형식

### 형식 1: 표준 형식
```
| No | Entity   | Name | Employee ID | Team          | Position | Start to work date |
|----|----------|------|-------------|---------------|----------|-------------------|
| 1  | Seoul HQ | 김철수 | EMP001      | Assembly Team | Operator | 2023-01-15        |
| 2  | Seoul HQ | 이영희 | EMP002      | Testing Team  | Tester   | 2023-02-20        |
```

**컬럼 설명:**
- **No**: 순번 (선택 사항, 무시됨)
- **Entity**: 법인/회사명 (필수)
- **Name**: 작업자 이름 (필수)
- **Employee ID**: 사원번호 (필수)
- **Team**: 팀/부서 (필수)
- **Position**: 직책/직위 (필수)
- **Start to work date**: 입사일 (필수)

### 형식 2: Work list 형식 (자동 변환)
```
| Name  | Employee ID | Company | Department  | Position    | start to work |
|-------|-------------|---------|-------------|-------------|---------------|
| 孙海亮  | NY392       | CSCN    | black tower | CNC Cutting | 2012-12-20    |
| 刘昌   | NY7027      | CSCN    | black tower | CNC Cutting | 2018-11-01    |
```

**컬럼 설명:**
- **Name**: 작업자 이름 (필수)
- **Employee ID**: 사원번호 (필수)
- **Company**: 회사명 → Entity로 자동 변환 (필수)
- **Department**: 부서명 → Team으로 자동 변환 (필수)
- **Position**: 직책/직위 (필수)
- **start to work**: 입사일 (필수)

## 날짜 형식

### 지원하는 날짜 형식
1. **텍스트 날짜**: `2023-01-15`, `2012-12-20`
2. **Excel 날짜**: Excel에서 자동으로 변환된 날짜 형식 (자동 처리됨)
3. **날짜/시간**: `2023-01-15 00:00:00` (시간 부분은 무시됨)

### 날짜 입력 권장 사항
- Excel에서 날짜 셀의 형식을 "날짜"로 설정
- YYYY-MM-DD 형식 사용 (예: 2023-01-15)
- 시스템이 자동으로 Excel 날짜를 변환함

## 업로드 방법

### 형식 1 사용 시
1. 작업자 등록 페이지 접속
2. 엑셀 파일 선택
3. '작업자 업로드' 버튼 클릭

### 형식 2 사용 시 (예: Work list_CN.xlsx)
1. 작업자 등록 페이지 접속
2. 엑셀 파일 선택 (시스템이 자동으로 형식 감지)
3. '작업자 업로드' 버튼 클릭

## 데이터 매핑

### 형식 2 → 시스템 내부 매핑
- `Company` → `Entity` (법인)
- `Department` → `Team` (팀)
- `start to work` → `Start to work date` (입사일)

## 예제 파일

### Work list_CN.xlsx
- 형식: Name, Employee ID, Company, Department, Position, start to work
- 작업자 수: 13명
- 특징: 중국어 이름, CSCN 법인, black tower 부서

## 오류 해결

### "필수 항목이 누락된 데이터가 있습니다"
- 모든 필수 컬럼에 데이터가 입력되어 있는지 확인
- 빈 셀이 없는지 확인
- 오류 메시지에 표시된 행 번호 확인

### "지원하지 않는 엑셀 파일 형식입니다"
- 컬럼명이 형식 1, 2와 일치하는지 확인
- 철자 오류 확인 (예: "Employee ID" vs "Employee_ID")
- 대소문자 구분 확인

### "날짜 형식 오류"
- Excel에서 날짜 셀의 형식을 "날짜"로 설정
- YYYY-MM-DD 형식 사용
- 날짜가 비어있지 않은지 확인

## 주의사항

### 중복 데이터
- Employee ID가 중복되면 기존 데이터를 업데이트합니다
- 동일한 사원번호로 재업로드하면 정보가 갱신됩니다

### 데이터 검증
- 모든 필수 항목이 입력되었는지 자동 검증
- 빈 행은 자동으로 건너뜁니다
- 날짜는 자동으로 변환됩니다

### 성능
- 한 번에 최대 1000명까지 업로드 권장
- 대량 데이터는 여러 번에 나누어 업로드

## 업로드 후 확인

1. 대시보드에서 전체 작업자 수 확인
2. 시험 응시 페이지에서 작업자 목록 확인
3. 올바른 정보가 표시되는지 확인
